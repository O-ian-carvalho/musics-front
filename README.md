# Sistema de Gerenciamento de Músicas

## Como Executar

Para rodar a aplicação localmente, use o seguinte comando:

```sh
ng serve
```

Acesse no navegador:
```
http://localhost:4200/
```

Para gerar os arquivos otimizados para produção:
```sh
ng build --prod
```
Os arquivos serão gerados na pasta `dist/`.
